# Project-WorkLink
## Workforce Management System

### Overview
Project-WorkLink is a Workforce Management System designed to manage
volunteers and temporary staff for events, exams, and short-term activities.
The system supports event creation, staff allocation, attendance tracking,
and payment processing through a centralized web platform.

### Technology Stack
- Frontend: React, HTML, CSS, Bootstrap
- Backend: Java, Spring Boot, Spring MVC
- Database: MySQL
- Tools: GitHub, Eclipse, Postman
